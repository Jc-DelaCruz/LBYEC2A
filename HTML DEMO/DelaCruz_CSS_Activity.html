  3. Differentiate the various types of web images
    • Joint Photographic Experts Groups (.JPG) - used for photographs, does not support transparency.
    • Graphics Interchange Format (.GIF) - a format for image that supports both animated and static.
    • Portable Network Graphics (.PNG) - used for images with flat colors and hard edges such as logo, illustrations and it supports transparency.
